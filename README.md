# 🚀 ATS Applicant Tracking System

A full-stack Applicant Tracking System (ATS) built using **React.js, FastAPI, and MySQL** to simplify the recruitment process through secure authentication, resume management, and candidate tracking.

---

## 📌 Overview

The ATS platform helps organizations manage candidate applications efficiently. It provides secure user authentication, resume uploads, candidate screening, and an admin dashboard for recruitment management.

---

## ✨ Features

### 👨‍💼 Candidate Module
- 🔐 User Registration & Login
- 📧 Email OTP Verification
- 👤 Profile Management
- 📄 Resume Upload
- 📊 Resume Analysis
- 📋 Application Tracking Dashboard

### 🛠️ Admin Module
- 🔑 Admin Authentication

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/rithikasundari/ATS-Applicant-Tracking-System.git
cd ATS-Applicant-Tracking-System
```

### 2️⃣ Backend Setup

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
